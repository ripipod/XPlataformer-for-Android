# XPlataformer Android Port

Yes, we've somehow ported [XPlataformer](https://github.com/ripipod/XPlataformer/) to Android!

### How did you port this?

We recoded the game in C++ to port this directly to Android.
> And yeah... we obviously added touch controlls and a Windows native C++ compilation of C++ which will be put on the normal repo of XPlataformer.

### In what version is the game based?

It was based on v1.0.0-beta4 (yes it does exist, but i never published it)
> This is also a test version of the java edition of XPlataformer

### Anything not on here?

The design was simplified to be able to run on limited devices with low CPU and RAM power
> The game is also compiled for Android 4.0 (API 18/19) on ARMABIv7a (32-bit) so don't expect it to not run on way too new versions of Android.
